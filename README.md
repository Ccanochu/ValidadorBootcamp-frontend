# Validador de Bootcamps - Frontend

---

## Técnologías utilizadas

- React
- Zustand
- Jest
- Vitest
- Vite
- Tailwind
- Typescript
- Axios

## Instalación

```bash
npm install
```

```bash
npm run dev
```
